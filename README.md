如何使用docker安装hexo
1. 安装 docker 和 docker compose
2. git clone https://github.com/zexinsg/docker-hexo.git
 
```
git clone https://github.com/zexinsg/docker-hexo.git
cd docker-hexo
bash restart.sh


默认使用的主题是fluid git地址: https://github.com/fluid-dev/hexo-theme-fluid
```
